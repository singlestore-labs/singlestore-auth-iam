SingleStore Cloud IAM Authentication API (0.1.0)

Download OpenAPI specification:Download

License: Proprietary

HTTP API for exchanging cloud provider identity credentials for SingleStore JWTs.

Customers may call these endpoints directly with their own HTTP client instead of using the s2iam client libraries (Go, Python, Java). Obtain cloud-provider credentials on the machine or workload making the request, then POST them to the authentication service using the headers documented below.

Production host: https://authsvc.singlestore.com

Status

  • Database (engine) JWTs: available; APIs may change before GA.
  • Management API JWTs: available; APIs may change before GA.

Related endpoints

  • JWT signing keys (JWKS): GET /auth/oidc/op/Customer/.well-known/jwks.json
  • Validate returned JWTs against this JWKS after exchange.

Cloud IAM credential exchange

Exchange cloud identity for a database (engine) JWT

Verifies the caller's cloud provider identity and returns a signed JWT suitable for SingleStore engine / database access.

Send an empty body. Authentication is performed exclusively via HTTP headers carrying cloud provider credentials (see components.securitySchemes).

The optional workspaceGroupID query parameter is sent by the reference Go client for database JWT requests. The server verifies identity from headers; callers using custom clients may include it for forward compatibility.

Authorizations:
awsCredentialsgcpBearerTokenazureBearerToken
query Parameters
workspaceGroupID
string
Example: workspaceGroupID=wg-0123456789abcdef

Target workspace group (reference client sends this for database JWTs).

Exchange cloud identity for a management API JWT

Verifies the caller's cloud provider identity, resolves a matching SingleStore cloud-principal user, and returns a signed JWT for management API access.

Send an empty body. Authentication is performed via cloud provider headers.

The verified identity must be linked to an active cloud-principal user in SingleStore; otherwise the server returns 404.

Authorizations:
awsCredentialsgcpBearerTokenazureBearerToken
